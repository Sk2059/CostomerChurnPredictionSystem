# 🚀 Customer Churn Prediction System

An end-to-end Machine Learning project that predicts whether a customer is likely to churn using multiple classification algorithms, feature engineering, hyperparameter optimization, and interactive Streamlit deployment.

---

# 📌 Project Overview

Customer churn is one of the biggest business challenges in telecom and subscription-based industries. This project uses Machine Learning techniques to analyze customer behavior and predict churn probability.

The project covers the complete ML lifecycle:

✅ Data Cleaning  
✅ Exploratory Data Analysis  
✅ Feature Engineering  
✅ Feature Encoding  
✅ Feature Scaling  
✅ Feature Selection  
✅ Multiple Model Training  
✅ Boosting Algorithms  
✅ Hyperparameter Tuning  
✅ Model Evaluation  
✅ Deployment with Streamlit  

---

# 🎯 Objectives

- Predict customer churn accurately
- Compare multiple ML algorithms
- Build production-style ML workflow
- Deploy interactive prediction system
